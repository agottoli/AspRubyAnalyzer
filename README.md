# Usage 
------------------------------------------

#### Install the environment.
Enter inside the _LivenessAnalysis_ and run the following command:

    omake install


#### Analyse a Ruby code.
Enter inside the _LivenessAnalysis/bin_ folder and run the following command:

    ./dls2 <nilness | liveness> <ruby_file>
